# time-o-clock
Time-o-clock is a simple project that show informations about the time. He is in the development phase.
He show you the date, hour, say "Bom dia", "Boa tarde" ou "Boa noite" to you, and tell you how many time have to the next minute, hour, week...
He haven't support to the English, this is going added in the next days.

Test he on: https://tys3ok.mimo.run/index.html
